# Components

React компоненты для Travel App.

## Структура

- `header/` - Верхняя панель с логотипом, профилем и настройками
- `search-form/` - Форма поиска маршрутов
- `navigation-tabs/` - Навигационные табы
- `assistant-button/` - Кнопка помощника (мамонтёнок)
- `routes-section/` - Раздел маршрутов с формой поиска
- `hotels-section/` - Раздел гостиниц

## Стиль

Все компоненты используют якутскую цветовую палитру:
- Бирюзовый (`yakutia-turquoise`)
- Голубой (`yakutia-blue`)
- Ледяной (`yakutia-ice`)
- Графит (`yakutia-graphite`)

## Использование

```tsx
import { Header, SearchForm, NavigationTabs } from '@/components'
```
